# Throwing-Distance-Calculator
throwdis コマンドを提供します。
物を投げたときの
	角度
	初速度
	投げた高さ
	地面の傾斜
から，重力加速度0.906 m/s においての
	到達する水平距離
	落下する時間
	落下したときの進行角度
を出力します.

# throwdisコマンド
[![test](https://github.com/peanutnuts/Throwing-Distance-Calculator/actions/workflows/test.yml/badge.svg)](https://github.com/peanutnuts/Throwing-Distance-Calculator/actions/workflows/test.yml)

## リポジトリの入手
リポジトリを保存したいディレクトリにて，
	$ git clone 
と入力してください。

## 使い方
	$ cd Throwing-Distance-Calculator
	$ python3 throwdis
と入力すると，以下の4つが標準入力に順番に表示されます。
	初速度 (m/s)を入力してください：
	投げる角度 (deg) を入力してください：
	高さ (m) を入力してください：
	地面の傾き (deg) を入力してください：
指示に従って，値を標準入力に半角数字で入力してください。
入力された値から計算を行い，以下を出力し，コマンドを終了します。
	到達距離: <計算結果> m
	落下時間: <計算結果> s
	落下時の進行角度: <計算結果> deg

もし，60s 以上になっても，仮想上の投擲物の座標が地面より下にならなかった場合，
以下の文章が出力されコマンドを終了します。
	飛翔時間が60秒を超過。計算を終了します。

※　投げる角度，地面の傾き，の二つについては，水平方向を0，そこから見上げる時は正の数，見下げる時は負の数で角度を入力してください。
※　地面の傾きについては 85(deg) ~ -85(deg) の範囲で入力してください。
※　重力加速度は9.806 m/s で固定です。
※　結果にはわずかながら誤差が生じます。
※　地面に到達するときの速度が早すぎると，結果の誤差が大きくなります。
